Nvidia Umumkan GPU Laptop GeForce MX450

KOMPAS.com - Nvidia pekan ini mengumumkan chip pengolah grafis ( GPU) baru untuk laptop, GeForce MX450. GPU yang ditujukan untuk produk laptop tipis ini merupakan suksesor dari Nvidia GeForce MX350 yang dirilis Februari lalu. Meski tergolong GPU entry level, GeFOrce MX450 dibekali dengan dukungan interface PCI Express (PCIe) 4.0 dan memori grafis jenis GDDR6 sehingga diklaim dapat memberikan peningkatan kinerja gaming, serta editing foto dan video. Baca juga: Kartu Grafis Nvidia GeForce RTX Super Meluncur dalam 3 Varian "(GeForce MX450) Turut mendukung teknologi Nvidia Optimus untuk memberikan keseimbangan antara daya tahan baterai dan performa," tulis Nvidia dalam sebuah laman berisi pengumuman GPU terkait.

Adapun dukungan software API yang diberikan mencakup DirectX 12, Vulkan API 1.2, OpenGL 4.6, serta Open CL 1.2, berikut teknologi Nvidia GPU Boost dan Cuda. Di luar spesifikasi singkat di atas, tak banyak informasi lain yang diungkap Nvidia mengenai GeForce MX450. Keterangan yang dihimpun KompasTekno dari Hexus.net, Jumat (28/8/2020) menyebutkan bahwa GPU ini kemungkinan akan mulai hadir di laptop pada Oktober nanti. Baca juga: AMD Radeon Pro VII Meluncur, Jegal Nvidia Quadro Sementara itu, situs Notebook Check menyebutkan bahwa GeForce MX450 yang memiliki kode nama "N18S-G5", "N18S-G5-B", dan "N18S-LP" tak lain menggunakan chip Turing 117 yang sama seperti GeForce GTX 1650, tapi dengan memory bus yang dikurangi menjadi 64-bit. Beberapa kode nama berbeda untuk GeForce MX450 disebut mengacu pada jenis varian berdasarkan frekuensi kerja dan tingkat TDP, masing-masing mulai 720 MHz hingga 1.575 MHz, dan12 watt hingga 28,5 watt
